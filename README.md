# Credit-Card-Financial-Dashboard
A data analytics finance dashboard that is made with the help of powerbi.

Credit Card Transaction Dashboard
Overview

This Power BI dashboard provides insights into credit card transactions, revenue generation, customer spending behavior, and transaction trends. It helps analyze customer activity across different card categories, expenditure types, education levels, and job profiles.

Key Metrics

Revenue: 55.32M
Total Interest Earned: 7.84M
Transaction Amount: 45M
Transaction Count: 656K

Dashboard Insights
Revenue by Card Category
Blue cards generate the highest revenue.
Silver, Gold, and Platinum cards contribute smaller portions.

Revenue by Expenditure Type
Top spending categories include:
Bills
Entertainment
Fuel
Grocery
Food
Travel

Revenue by Customer Job

Major revenue contributors:

Business professionals
White-collar employees
Self-employed customers
Government employees
Revenue by Education Level
Graduates contribute the highest revenue.
High school and unknown education groups follow.

The dashboard tracks:
Quarterly revenue trends
Total transaction volume across Q1–Q4

Filters Available
Users can filter data by:

Quarter (Q1–Q4)
Gender
Card Type
Week Start Date


Tools Used

Power BI
Data Visualization
DAX Measures
Interactive Filters & Slicers

Purpose
This dashboard is designed to help businesses:

Monitor customer spending patterns
Identify high-value customer segments
Analyze revenue performance
Support data-driven financial decisions
